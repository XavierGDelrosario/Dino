Beyond Features
Feature: Word map, multidimensional showing distance between words in similar contexts and difficulty for the average user. Allows for features like suggested words.
Concerns: Technical concern, how to make this, is there any public data on this. Needs to scale to with languages added

Feature: User language level calibration.Quizzes the user with a paragraph to determine around what level they are.

Feature: Media Translation, Online videos(tiktok, Instagram etc), Books, Manga, Tv shows, Movies, etc. Allows the user to study a bit before using media.
Concerns: Legal, for something like scripts we might not have access. How to obtain the info and be able to put it through our translation. Accuracy. API costs.

Feature:Multiple language support
Concerns: Uncertain if a dictionary exists between 2 languages. Adding a third language doesn’t guarantee all three can cross with each other. Using a language train is risky because it's a game of telephone. Strictly falling back to MT loses accuracy. For languages like Russian, Chinese, Korean, make sure the reading is done properly.

Feature:Different input methods:camera, voice, drawing
External API: Google
Concerns: Pricing is usage based, maybe limit or completely lock these features unless the user is subscribed (adds valid payment,subscription, and security concerns).

Feature: Hosting independent creators to add their own media, lessons for language learning.
Concerns: Copyright, inappropriate content, needs a good existing infrastructure on analytics. Secure payment for paying creators. How to populate app with creators and users(advertisement issue)
